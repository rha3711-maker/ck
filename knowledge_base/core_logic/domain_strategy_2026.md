# 2차 도메인 및 무한 확장 전략 (Digital Real Estate)

> 본 내용은 [애드센스팜의 도메인 및 하위 도메인 로직 칼럼](https://adsensefarm.kr/%eb%8f%84%eb%a9%94%ec%9d%b8-%eb%9c%bb%ea%b3%bc-1%ec%b0%a8-2%ec%b0%a8-%ed%95%98%ec%9c%84%ec%84%9c%eb%b8%8c-%eb%8f%84%eb%a9%94%ec%9d%b8-%ea%b0%9c%eb%85%90-%ec%99%84%eb%b2%bd-%ec%a0%95%eb%a6%ac/)의 핵심 로직을 정리한 지식 베이스입니다.

---

## 1. 2차 도메인의 중요성
- **개요**: 플랫폼 주소(tistory.com/xxx)가 아닌 본인만의 고유 주소(domain.com)를 사용하는 것.
- **가치**: 
    - **브랜드화**: 사용자에게 신뢰를 줌.
    - **자산 소유권**: 플랫폼이 사라지거나 정책이 변해도 도메인은 본인의 소유로 유지됨.
    - **애드센스 승인 효율**: 루트 도메인 승인 시 하위 도메인은 별도 승인 없이 광고 송출 가능.

---

## 2. 하위 도메인 (Sub-domain) 무한 확장 전략 💎

### 2.1. 승인 우회 로직
- `example.com` (루트 도메인)으로 애드센스 승인을 한 번 받으면:
- `blog1.example.com`, `news.example.com`, `app.example.com` 등 하위 도메인들을 추가 승인 없이 즉시 수익화할 수 있습니다.
- 이를 통해 수십 개의 테마별 사이트를 빠르게 구축하여 수익을 폭발시키는 구조가 가능합니다.

### 2.2. 사이트 분리 (Risk Management)
- 특정 주제(예: 도박성 이슈, 민감한 정치 등)가 저품질에 걸리더라도 하위 도메인 단위로 리스크를 격리할 수 있습니다.

---

## 3. 기술적 세팅 포인트 (Technical Setup)
- **가비아(Gabia), 호스팅어(Hostinger), 클라우드플레어(Cloudflare)** 등에서 도메인 구매 및 관리.
- **DNS 설정**: CNAME 레코드를 통해 티스토리나 구글 블로거에 연결.
- **HTTPS 보안 연결**: SSL 인증서 적용 필수 (클라우드플레어 활용 시 무료).

---

## 📝 작전 지침 (Kodari Tip)
- **디지털 부동산을 소유하라**: 남의 땅(플랫폼 기본 주소)에 집을 짓지 마십시오. 도메인은 디지털 세상에서 본인의 등기부등본과 같습니다.
- **멀티 사이트 전략**: 하나의 블로그에 모든 주제를 섞지 말고, 하위 도메인을 활용해 주제별 전문 사이트를 여러 개 운영하여 검색 엔진 신뢰도를 높이십시오.
