# 블로그 포스팅 마스터 전략 (rha3714)

이 문서는 건강/비즈니스 블로그 운영의 핵심 운영 원칙과 포맷 가이드를 담고 있다.

## 1. 기본 운영 원칙
- **목표**: 정보성 콘텐츠를 통한 트래픽 유입 및 CPS 수익화.
- **톤앤매너**: 친근하면서도 전문적인 1인칭 시점 (예: ~입니다, ~해 보세요).
- **공정위 문구**: CPS 링크 포함 시 **본문 최상단**에 일반 텍스트로 반드시 삽입.
  이 포스팅은 네이버 쇼핑 커넥트 활동의 일환으로, 판매 발생 시 수수료를 제공받습니다.

## 2. 콘텐츠 구조 (Formatting)
- **제목**: 호기심 자극, 숫자 활용, 전문가 권위 활용 (알파남 진실 로직).
- **본문**: 짧은 단락, 이모지 적극 활용, 가독성 중심.
- **기호 금지**: 불렛 포인트(-), 마크다운 리스트(1.) 등 모든 기호 사용 금지.

## 3. 최종 전달 가이드 (All-in-One Code Block)
- **복사 최적화**: 사용자가 네이버 에디터에 붙여넣었을 때 줄바꿈과 서식이 깨지지 않도록, **포스팅 본문 전체(공정위 문구~내부 링크까지)**를 하나의 **마크다운 코드 블록(```text)** 내에 작성하여 제공한다.
- **이모지 활용**: 코드 블록 내에서도 이모지를 풍부하게 사용하여 시각적 효과를 극대화한다.

## 4. 섹션별 상세 포맷
### [추천 상품 섹션]
- 상품명
- 상품에 대한 설명 (엔터로 줄바꿈)
- 항목 간 한 줄 공백.

### [함께 읽으면 좋은 정보]
- 제목 URL (한 줄에 나란히 작성, 항목 간 한 줄 공백)

## 5. 이미지 전략 (Genspark)
- 한국인 모델 중심, 밝고 건강한 느낌, 썸네일용 여백 확보 프롬프트 사용.
